# Инструкция для спеца — вставка в Tilda T123

**Страница:** walk-walk.ru/intervals

## Что вставлять

Нужно **3 блока T123** (HTML-код) на странице, ПОДРЯД друг за другом, в правильном порядке:

| Порядок | Файл | Размер | Что внутри |
|---------|------|--------|-----------|
| 1 | `tilda-block-1.html` | 23 KB | Подключение шрифтов/CSS + шапка + мобильное меню + Hero + Оффер + Метод + **попапы оплаты** + sticky CTA + back-to-top |
| 2 | `tilda-block-2.html` | 21 KB | Бонус (клуб) + Тарифы (3 GetCourse-виджета) + Лицензия |
| 3 | `tilda-block-3.html` | 28 KB | Кейсы + отзывы (marquee) + FAQ + Footer + подключение JS |

Все три файла **≤ 28 KB** (лимит Tilda 30 KB).

## Почему 3 блока

1. Полный HTML-код страницы — ~140 KB, в один блок T123 не помещается (лимит 30 000 символов).
2. Кроме того, **вся кириллица закодирована как HTML-сущности `&#NNNN;`** — это лечит mojibake (когда Тильда пересохраняла блоки и буквы превращались в «РёР°РіР°Р№»). Кодировка раздувает файлы в ~2 раза, поэтому 3 блока.
3. Попапы оплаты вынесены в блок 1 (они `position: fixed`, место в DOM не важно), чтобы блок 3 влез.

## Почему mojibake был

В прошлый раз в Tilda T123 русский текст рендерился как «РёР°РіР°Р№» вместо «Шагай». Это значит, что Тильда сохранила блок в одной кодировке, а отдала в другой.

Решение: в новых блоках вся кириллица **закодирована в HTML-сущности** (`&#1064;&#1072;&#1075;&#1072;&#1081;` = «Шагай»). Это чистый ASCII — Тильда не может его испортить никакой кодировкой, браузер при отображении соберёт обратно русские символы.

## CSS, JS и картинки

- Грузятся по абсолютным URL с **GitHub Pages**: `https://npopko55-cmd.github.io/intervals/`
- Никаких локальных загрузок в Tilda не требуется
- Cache-busting через `?v=intv-G` — при правках просто меняется версия, спецу повторно вставлять блоки не нужно

## Дополнительные настройки в Tilda

1. **Метрика 94057307** — подключить на странице через стандартный механизм Тильды (Настройки сайта → Аналитика → Yandex.Metrika). Цели полетят автоматически через `data-metrika-goal` атрибуты.
2. **UTM-метки** — пробрасываются автоматически через JS на все ссылки `walk-walk.ru` (в том числе на кнопки GetCourse-виджетов, которые подгружаются динамически).
3. **3 блока должны идти ПОДРЯД** на странице — между ними ничего не вставлять (иначе sticky-CTA и попапы могут оказаться внутри неправильного контейнера).

## Если нужно обновить

Запустить пересборку локально:
```bash
cd intervals
python3 build-tilda-split.py
```

Три файла обновятся — спецу заново скопировать содержимое в Tilda T123 блоки.
